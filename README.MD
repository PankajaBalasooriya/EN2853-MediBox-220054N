## ğŸš€ Getting Started

### Prerequisites
- Arduino IDE with ESP32 board support
- Node-RED installation
- MQTT broker access (test.mosquitto.org)

   - Upload the Arduino sketch to ESP32
   - Deploy the Node-RED flow
   - Access the dashboard through Node-RED interface
